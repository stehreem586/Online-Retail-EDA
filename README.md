# Online Retail II - Exploratory Data Analysis (EDA)

##  Project Overview

This project performs Exploratory Data Analysis (EDA) on the Online Retail II dataset. The objective is to understand customer purchasing behavior before building a recommendation system.

The analysis includes:
- Dataset exploration
- Missing value analysis
- Duplicate detection
- Top-selling products
- Country-wise sales analysis
- Monthly revenue trend
- Correlation analysis
- Outlier detection
- Business insights

---

##  Dataset

**Dataset Name:** Online Retail II

**Source:** UCI Machine Learning Repository

The dataset contains online retail transactions from a UK-based online retailer between 2009 and 2011.

##  Analysis Performed

- Loaded and explored the dataset
- Displayed dataset shape, columns and data types
- Identified missing values
- Identified duplicate rows
- Created Revenue column
- Top 10 products by Quantity
- Top 10 products by Revenue
- Sales analysis by Country
- Monthly Revenue Trend
- Correlation Heatmap
- Outlier Detection using Boxplots

---

##  Business Insights

1. The United Kingdom generated the highest revenue.
2. A few products contribute significantly to total sales.
3. Revenue shows seasonal trends across months.
4. Several outliers exist in Quantity, Price and Revenue.
5. Missing Customer IDs indicate incomplete customer information.
